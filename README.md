# Dominion cal

![](./doc/img/dominion_cal.gif)

## What is it?
* 山札から5枚のカードを引いて手札にしたとき、その手札から何金出せるかの期待値を計算するアプリです。
* 現時点では+アクション,+ドロー,+金するアクションカードにしか対応していません。(礼拝堂や鉱山のようなカードには対応していません)
* まだ作りかけです

## Usage

require ionic

```
$ npm install -g ionic cordova
```

run in browser

```
$ ionic serve
```

or 

```
$ ionic cordova run browser
```
